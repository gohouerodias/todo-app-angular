TodoApp — это веб-приложение на Angular для управления напоминаниями (todo).

Приложение содержит:

Страницу со списком напоминаний

Одну страницу для создания и редактирования напоминаний

Установка
git clone <https://github.com/gohouerodias/todo-app-angular>
cd TodoApp
npm install

Запуск
ng serve


Открыть в браузере:

http://localhost:4200/

Тестирование
ng test
ng e2e
